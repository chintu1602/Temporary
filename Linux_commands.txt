Linux
kernel, shell , hardware layer, utilities

kernel :
main core component in Linux, it control the activity of other hardware components.
process in queue
types:
1. monolithic kernel 
--- concurrent process executed simultaneously
--- all process share same memory locations
2. micro kernel
--- user services and kernel services are executed in separate address space
3. exokernel
--- it helps to manage hardware resources at application level
4. hybrid kernel
---combination of above monolithic and micro kernel

Linux kernel subsystem
1. Process Scheduler
2. Memory management unit (MMU)
3. Virtual File system (VFS)
4. Networking Subsystem
5. Process Communication Unit

copying and pasting files and directories

1. cp --- copying and pasting files or directories 
    cp [option] source destination

i -- interactive mode. prompts before overwriting files
-r or -R ---recursive mode , it will copy directories and its content
-u -----copy only when the source file is newer tahn the destination file 
	or when the destination file is missing
-p -----preserved file attributes
-v ---- verbose mode... it shows the progress of files being copied
-a --- archive mode

rm -rvf ---- for removing files and directories

Renaming/moving files and directories

mv --- similar to cp but it destroys the source file
	mv [option] source destination

-i ------ interactive mode
-u ------ move when the source file is newer than the destination file or when the destination file is missing
-v ------ verbose mode ....it shows the progress of files being moved
-f ----- force. overwrite destination files without prompting
